# Balatro Mod Sort

Sort your Balatro Mods Menu with multiple options — without messing up the load order!

## Sort Options

- **A→Z** — Alphabetical sort by display name (default)
- **Recently Updated** — Sort by most recently modified mod files (newest first)
- **Config Changed** — Sort by most recently changed mod config files (newest first)

Click the **sort button** at the top of the Mods menu to cycle between sort modes.

## Installation

Requires [Steamodded](https://github.com/Steamopollys/Steamodded) 1.x.

Drop the mod folder into your Balatro `Mods` directory.
